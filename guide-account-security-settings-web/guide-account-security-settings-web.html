帳戶安全設置

註冊與登入帳號

使用電子郵件或手機號碼完成註冊，並設定高強度密碼。

登入時，請避免使用公共 Wi-Fi 或不安全的設備，以降低帳號被入侵的風險。

⚠️ 注意：任何人索取您的密碼或驗證碼皆可能是詐騙，請勿提供。

KYC 身份驗證

為保障資金安全並符合監管要求，請完成KYC（Know Your Customer）身份驗證。

通過驗證後，可提升交易與提領額度，並開啟更多功能。

您的身分資料將依法保密，僅用於合規與安全目的。

雙重驗證

建議啟用雙重驗證，如 Google Authenticator 或簡訊、郵箱驗證碼。

這能在登入、提幣或修改安全設定時提供額外的保護層，確保僅有您本人能操作帳戶。

資金密碼

在帳戶安全中設定資金密碼，在進行提幣、轉帳或高風險操作時，需額外輸入此密碼。

此機制可有效防止帳號被入侵後，資金直接被他人操作轉移。

登入記錄查詢

您可以在「帳號活動」中查看最近的登入時間、地點與 IP 位址。

若發現 異常登錄（例如未知的地點或設備），請立即修改密碼並聯絡客戶服務。

防釣魚提醒

請務必認明 官方網址，警惕索取密碼或驗證碼的釣魚網站與郵件。

⚠️ 提醒：官方客服絕不會主動向您索取密碼、私鑰或要求轉帳至個人錢包。
